Donkey Bridge

A vocabulary app for visual learners

Overview

This is a case study about an ux design concept created in spring 2020.

My role: UX-Designer

Challenge

How might we help visual learners to memorise new vocabulary quickly on the go.

Goal

Creating a vocabulary learning app which supports the easy memorising of vocabulary in a very visual way.

Process Steps

  • Competitor Research
  • User Research (User Interviews)
  • User Persona
  • Task Analysis
  • Crazy 8's Technique
  • Low Fidelity Prototyping
  • User Testing
  • High Fidelity Wireframes

Tools

  • Adobe Comp App (Ipad)
  • Marvel App
  • Whimsical
  • Pen&Paper&Postits
  • Voice Recorder App
  • Videocamera App
